# GCE™ - Global Context Engine™

**The first AI prompt system that knows where you are.**

Auto-injects **real-time local context** into any prompt:
- `{{city}}` → Malé
- `{{currency}}` → MVR
- `{{time}}` → 12:58 PM
- `{{temp}}` → 31°C
